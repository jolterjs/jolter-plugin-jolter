# Security Policy

## Supported Versions

Security fixes are made on the latest released version of this plugin.

## Reporting a Vulnerability

Please report security issues privately to security@jolter.dev. Do not open a
public issue for vulnerabilities.

Include:

- affected version;
- reproduction steps or proof of concept;
- expected impact;
- any relevant logs or release artifact names.

The maintainers will acknowledge reports and coordinate remediation.
